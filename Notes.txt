###### SET UP #######

1. Go to Raspberry Pi Configuration - Interfaces
	Enable SSH

2. Go to Raspberry Pi Configuration - Display
	Disable screen blanking (stop going to sleep)

3. To hide the mouse cursor from the screen install unclutter
	$ sudo apt install unclutter sed

4. Transfer pi-player folder to user/Documents

5. Convert start_server.sh from DOS to UNIX (to remove hidden \r carriage returns which causes cd command to fail)
	You may need to download the software with this command:
	$ sudo apt-get install dos2unix
	
	a. $ cd Documents/pi-player
	b. $ dos2unix start_server.sh

6. In Terminal. Set start_server.sh script to run at start up.
	$ sudo nano /etc/rc.local
	Scroll down to the line above exit 0
	Enter the following line.
		sh /home/user/Documents/pi-player/start_server.sh
	Crl X
	Y
	Enter

7. Chromium - Settings - System 
	Turn off - Continue running background apps when Chromium is closed
	Turn off - Use hardware acceleration when available

8. Chromium - Settings - Extensions
	Turn off all extensions

9. Chromium - More Tools - Developer Tools
	On Network tab select the Disable Cache tick box.

10. Hyper Pixel 4.0 square display
	$ sudo nano /boot/config.txt

	Add following lines at bottom
		# Hyper Pixel 4.0 Display
		dtoverlay=vc4-kms-dpi-hyperpixel4sq

	To rotate
		dtoverlay=vc4-kms-dpi-hyperpixel4sq,rotate=90

####
If sudo commands from Peerless Player fail with error :: no tty present and no askpass program specified.
	In Pi edit file: sudo nano /etc/sudoers
	And add at end of file:
		# Allow members of group sudo to execute any command
		%sudo   ALL=(ALL:ALL) ALL
		userName ALL=(ALL) NOPASSWD: ALL

SWITCHING SCREENS

For 8" screen comment edit below lines in config.txt file

$ sudo nano /boot/config.txt

# Enable DRM VC4 V3D driver on top of the dispmanx display stack
dtoverlay=vc4-fkms-v3d
max_framebuffers=2

[all]
#dtoverlay=vc4-fkms-v3d
ignore_lcd=1
dtoverlay=vc4-kms-v3d
dtparam=i2c_vc=on
dtparam=i2c_arm=on
dtoverlay=WS_xinchDSI_Screen,SCREEN_type=8,I2C_bus=10
dtoverlay=WS_xinchDSI_Touch,I2C_bus=10,invertedx,swappedxy
